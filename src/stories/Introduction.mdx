# Amateur Cycling Stats - Design System

Welcome to the **Amateur Cycling Stats Design System**, built with [shadcn-svelte](https://shadcn-svelte.com) and [Tailwind CSS 4](https://tailwindcss.com).

## Overview

This design system provides a comprehensive set of reusable components for building consistent, accessible, and beautiful user interfaces in the Amateur Cycling Stats application.

### Key Features

- ✅ **21 shadcn-svelte Components** - Production-ready UI primitives
- ✅ **Svelte 5 Compatible** - Uses latest runes syntax ($props, $state, $derived, $effect)
- ✅ **Tailwind CSS 4** - Modern utility-first CSS with OKLCH color space
- ✅ **Dark Mode Support** - Complete dark theme with seamless toggling
- ✅ **Fully Typed** - TypeScript strict mode throughout
- ✅ **Accessible** - WCAG 2.1 compliant components with proper ARIA attributes
- ✅ **Internationalized** - i18n-ready with sveltekit-i18n integration

## Getting Started

### Installation

All shadcn-svelte components are already installed. To use a component:

```typescript
import { Button } from '$lib/components/ui/button';
import { Card, CardHeader, CardTitle, CardContent } from '$lib/components/ui/card';
```

### Usage Example

```svelte
<script lang="ts">
  import { Button } from '$lib/components/ui/button';
  import { Card, CardHeader, CardTitle, CardContent } from '$lib/components/ui/card';

  let count = $state(0);
</script>

<Card>
  <CardHeader>
    <CardTitle>Counter Example</CardTitle>
  </CardHeader>
  <CardContent class="flex flex-col gap-4">
    <p>Count: {count}</p>
    <Button onclick={() => count++}>
      Increment
    </Button>
  </CardContent>
</Card>
```

## Svelte 5 Runes

All components follow Svelte 5 best practices:

- ✅ Use `$props()` for component props
- ✅ Use `$state()` for reactive state
- ✅ Use `$derived()` for computed values
- ✅ Use `$effect()` for side effects
- ✅ Use `{@render children()}` for slot content
- ❌ Never use `export let` (Svelte 4 syntax)
- ❌ Never use `<slot />` (deprecated)
- ❌ Never use `$:` reactive statements
